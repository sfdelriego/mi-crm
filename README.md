# VideCRM

Landing page de presentación para un CRM moderno, construida en HTML y CSS puro, sin dependencias.

## Características

- **Gestión de clientes** — pipeline visual de ventas, historial 360° y automatizaciones
- **Redes sociales y leads** — bandeja unificada de Instagram, LinkedIn, X y TikTok con captación automática de leads mediante IA
- **Analytics** — métricas en tiempo real, previsión de ingresos y KPIs clave (Churn, NPS, CAC, LTV)

## Tecnologías

- HTML5
- CSS3 (glassmorphism, gradientes, animaciones)
- Sin frameworks ni dependencias externas

## Uso

Abre `index.html` directamente en el navegador.

## Capturas

La página incluye mocks interactivos de UI que simulan:
- Lista de contactos con etiquetas de estado
- Pipeline de ventas con importes
- Bandeja de mensajes sociales
- Gráfica de ingresos semanales
